    App::testyourears

    This archive contains the distribution App:testyourears, version 0.001. This is a program testyourears that generates random tones in one ear to be matched by a tone sent to the other ear. The discrepancies are reported in a file which can later be read or plotted.

    Read the file INSTALL for installation instructions.

    Read lib/App/testyourears.pm for usage instructions and other information.
